AgriSense: An intelligent, hardware-optimized agricultural field monitoring system

Developed low-cost wireless Bluetooth Low Energy and Wi-Fi based agricultural field monitoring system that tracks soil and environmental conditions in real-time using fuzzy logic and linear regression algorithms. Firmware was developed using C language and Apache NimBLE library. Hardware comprised Espressif ESP32-C3 and ESP32-S3 microcontrollers, Sensiron SHT41 temperature and humidity sensor, Rohm Semiconductor BH1750 light sensor, and capacitive soil moisture sensor.
